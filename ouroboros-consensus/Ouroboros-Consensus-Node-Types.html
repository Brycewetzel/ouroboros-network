data CandidateDecline Source #

A reason that we might choose to ignore a CandidateFragment.

Constructors

DeclineNotPlausible

The candidateChain is not better than our current selected chain.

DeclineNull

The candidate is empty (which should only happen if candidateIsLowDensity).

DeclineStale

The candidate does not even intersect the current selected chain. This can happen if BlockFetch has re-checked before ChainSync has had a chance to update the candidate.
